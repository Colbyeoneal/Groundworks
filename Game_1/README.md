# Groundworks

A simple Python incremental game inspired by Universal Paperclips.

## Run the game

```bash
python3 game.py
```

## How to play

- `make` — produce widgets.
- `buy` — improve your production power.
- `machine` — unlock a machine for bonus widgets.
- `auto` — enable automation when available.
- `research` — invest in AI improvements at later levels.
- `status` — view your current progress.
- `help` — show the command list.
- `quit` — exit the game.

Grow your widget production, level up through new stages, and reach the final level to win.
